- 사이트 목적
    - 여행 정보 사이트
    - 명소, 음식, 숙박 + 축제 등 4가지 카테고리를 통해 1차분류 / 다양한 데이터 활용 방식으로 접근성을 높임
        - 위 4가지 데이터를 묶어 여행코스라는 새로운 카테고리로 활용 가능
        - 검색 기능을 통해 원하는 정보를 쉽게 찾을 수 있도록 제공
        - 사용자들이 리뷰와 평점을 남길 수 있는 기능을 추가하여 신뢰성 있는 정보 제공
        - 현실적인 여건은 무시하고 전체 결제 및 예약기능 추가(고민할부분)
    - 사용자는 카테고리의 대/중/소분류를 통해 다양한 방식으로 데이터에 접근 가능
        - 1차적으로 명소, 음식, 숙박, 축제의 대분류를 통한 아이템 접근 가능
        - 2차적으로 랜덤추천, 광고, 인기순추천, 지역별추천, 최신순 추천 등을 통한 분류를 통해 아이템 접근 가능
        - 해쉬태그, 품목이름, 품목내용, 시기 등등 다양한 카테고리를 통한 검색기능 구현

    - 따로 정하진 않았으나 지나치게 올드한 분위기는 지양
- 디자인 컨셉
    - 로고 및 로고 이미지 확정 → white&green
    
    - 녹색과 흰색기반으로 전체적인 이미지를 조성하되 이미지 파일의 컬러와 분위기를 통해 전체적인 이미지 조정
- 공통
    - 페이지네이션
        - 기본적으로 10개까지 조회 가능
            - 여행상품품목은 최대 30개까지 출력,
            - 페이지리스트는 10개까지 출력
        - 페이지리스트는 한번에 5개까지 출력됨
        - 페이지리스트는 앞뒤로 2개가 출력
        - 이전, 다음 버튼을 통해 한번에 5개의 페이지를 넘길 수 있음
        - 가장 최근, 가장 오래된 페이지로 버튼을 통해 갈 수 있음
- 사용자기능
    - 회원가입
        - 이메일 회원가입
            - 이메일인증 절차 필요
                - 회원가입시 기입한 이메일로 인증메일 발송
                - 하나의 이메일당 한명의 회원만 인증 및 가입가능
                - 이메일 인증시 60분 이내에 인증필요(시간 수정예정)
                - 이메일 인증시 영어,숫자가 포함된 무작위 8자리 코드 생성 후 발송
                - 회원가입시 이메일 인증이 선행되어야 회원가입 진행 가능
        - SNS 회원가입
            - 하나의 SNS계정당 한명의 회원만 인증 및 가입 가능
            - SNS인증시 60분 이내에 인증 필요
            - 주소를 추가입력후 가입
    - 필수 정보입력
        - 필수정보가 입력되지않으면 회원가입 진행 불가능
        - 아이디
            - 영어대소문자와 숫자를 포함한 6~12자리로만 생성가능
            - 중복아이디 생성불가
                - 아이디 중복체크를 통과해야만 회원가입 진행 가능
                - 중복체크후 아이디 변경시 다시 중복체크 실행필수
        - 비밀번호
            - 영어대소문자와 숫자, !@#$를 포함한 8~16자리로 생성가능
            - 연속된 문자는 4번이상 사용불가
        - 비밀번호 확인
            - 비밀번호와 중복 체크 ⇒ 중복 시 O
            - 실패 시 ⇒ 실시간으로 아래에 붉은 글씨로 알림, 회원가입 진행불가
        - 이름
            - 1자~13자까지 입력
        - 생년월일
            - 생년월일 8자리 yyyy-MM-dd 입력
        - 휴대전화
            - 하나의 번호로 하나의 계정에 가입 가능
            - 앞 3자리 옵션으로 선택 후 뒤에 7~8자리 입력
    - 선택 시 입력
        - 주소
            - 실시간 주소조회 API를 이용하여 주소 입력
        - 성별
            - 라디오박스로 체크 남/여
    - 닉네임배정
        - 회원가입 완료후 자동으로 생성 후 배정
        - [user-]+[A-Z0-9]{16}의 랜덤으로 생성
        - 하나의 닉네임은 한명의 회원만 사용 가능
- 비회원 기능
    - 여행 상품 조회
        - 자신이 본 상품 최근 3개까지 조회 가능(사이드바 퀵메뉴)
        - 해당 목록을 클릭하여 곧바로 해당 게시물의 상세페이지로 이동
        - 상세페이지에서 예약,결제 등 추가 기능 사용시 로그인 페이지로 이동
        - 검색기능
            - 상세리스트에 담긴 품목들 검색가능
            - 검색어 길이에는 제한이 없음
        - 상세게시글 조회 가
- 회원기능

        - 아이디와 비밀번호를 입력하여 일치할 시 로그인 가능
        - SNS로그인이 되어있을시, SNS로그인버튼을 통해 로그인 가능

            - 로그인 유지는 사용자 미갱신시 최대 90일까지 유지됨
        - 로그인 실패시 안내메시지를 출력
    - 아이디 찾기
        - 회원가입시 입력했던 이메일, 이름, 생년월일 입력시 진행 가능
        - 해당 이메일로 마지막 3자를 ***로 표시한 아이디  발송
    - 비밀번호 찾기
        - 회원가입시 입력했던 이메일, 아이디, 이름, 생년월일 입력시 진행 가능
        - 비밀번호를 무작위 [a-zA-Z0-9]{16}로 변경
        - 변경된 비밀번호를 해당 이메일로 발송
        - 비밀번호 초기화 진행 후 첫 로그인시 비밀번호를 변경하라는 알림메시지 출력
